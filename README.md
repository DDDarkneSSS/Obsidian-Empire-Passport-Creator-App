# Obsidian-Empire-Passport-Creator-App
Приложение для простого и удобного создания паспорта

Новое приложения для создания паспортов, написанное при помощи Deepseek и DuckDuckGo AI chat-gpt-5 (я не кодер, но и запрягать других работать за бесплатно, я не собираюсь).

Было написано примерно за неделю и использует для работы ваш браузер. Приложение обязательно открывать через файлик "Obsidian Empire Passport Creator App.exe", который запустит локальный сервер на вашем компьютере и вкладку с интерфейсом приложения в вашем браузере (по умолчанию):
Проще говоря, откроется два окна: одно - с интерфейсом приложения (в веб-браузере по умолчанию), второе, запускающее локальный сервер http://localhost:3000 персонально на вашем компьютере. Если закрыть localhost, то приложение перестанет работать, так что оставьте его открытым до конца создания вашего паспорта. После завершения создания, сохраните готовый паспорт, нажав на "Сохранить PNG 1600x900". Сохранённый PNG будет лежать у вас в папке "Загрузки" м названием passport-1600x900 - *номер файла*.

Функции приложения:
* Возможность быстро и просто заполнить паспорт;
* Смена цвета и переключение светлой и тёмной темы;
* На кнопку "ℹ" можно открыть чеклист и посмотреть минимальные требования к паспорту;

Настоятельно не рекомендуется:
* Модифицировать приложение;
* Изменять положение внутреннего содержимого приложения.

FAQ:

В: У меня не отображается правая часть приложения, что делать?
О: Правая часть приложения не вмещается в экран, просто уменьшите масштаб веб-страницы ([ctrl]+[-] или [ctrl]+[колёсико мыши вниз]), или увеличьте размер окна браузера.

В: У меня перестал грузиться интерфейс, что делать?
О: Вероятно, вы закрыли .exe файл, который отвечал за хостинг приложения на локальном сервере (localhost), откройте его заново, нажав на файл .exe или start.bat, и перезагрузите страницу.

В: Я закрыл приложение, в процессе создания паспорта, могу ли я восстановить созданный паспорт?
О: К сожалению, такой функции в приложении нет и, вероятно, не предвидится.

В: Будет ли функция подвязки информации напрямую из Дискорд-бота Непупы (.lara) и ролей на Дискорд сервере?
О: Нет, не будет. Такая функция, вероятно, потребует подвязки Дискорд АПИ, что довольно сложно, в понимании, для меня. Кроме того, приложение преследует цель стать оффлайн-помощником в создании паспорта.

В: Помнится, раньше тоже был сервис для создания паспорта, но в виде сайта (https://passportoe.ru/). Почему в этот раз веб-приложение?
О: Сайт требует много ресурсов, должен иметь хост, который потребляет деньги, взамен на предоставление неокупаемой услуги создания картинки. Веб-приложение же, требует лишь наличие браузера и запускает сервер прямо на вашем компьютере, что сокращает затраты на содержание хост-машины.

В: КАК СКАЧАТЬ???
